# 学员手册

欢迎参加 OpenClaw 初学者课程。

## 课程形式
- 预备课（45–60 分钟）
- 第 1 课（60 分钟）
- 第 2 课（60 分钟）
- 第 3 课（60 分钟）

## 学习目标
课程结束后，你可以：
- 稳定运行 OpenClaw，
- 完成带工具调用的工作流，
- 设计一个可复用的真实场景工作流。

## 必备条件
- 电脑 + 稳定网络
- 已安装并可运行 OpenClaw
- Telegram（用于课程沟通）
- GitHub 账号
- LLM 访问能力（API Key / 订阅 / 课程统一提供）

## 卡住时如何求助（Stuck Protocol）
请按以下格式提交：
1. 目标
2. 已尝试步骤
3. 实际输出/报错
4. 预期结果
5. 你的判断
