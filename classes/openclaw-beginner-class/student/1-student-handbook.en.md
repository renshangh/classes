# Student Handbook

Welcome to the OpenClaw Beginner Program.

## Program format
- Prep Class (45–60 min)
- Class 1 (60 min)
- Class 2 (60 min)
- Class 3 (60 min)

## What success looks like
By the end of the program, you can:
- run OpenClaw reliably,
- complete tool-enabled workflows,
- build one repeatable real-world workflow.

## Required setup
- Computer + stable internet
- OpenClaw installed and running
- Telegram access for class communication
- GitHub account
- LLM access (API key/subscription or class-provided access)

## How to get help (Stuck Protocol)
Share:
1. Goal
2. What you tried
3. Actual output/error
4. Expected output
5. Your best guess
