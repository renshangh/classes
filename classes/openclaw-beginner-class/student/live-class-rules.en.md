# Live Class Rules (Student)

## Three modes
1. Lecture Mode: keep questions to logistics or urgent class-wide blockers
2. Practice Mode: ask for help using the Stuck Protocol
3. Q&A Mode: open questions and deeper discussion

## Where to post
- Meet: live voice questions
- Telegram: quick support and blockers
- GitHub: homework and issue tracking

## If you are stuck, post:
1. Goal
2. What you tried
3. Actual output/error
4. Expected output
5. Your best guess

## Support style
TAs will help in steps:
- Hint
- Scaffold/template
- Full walkthrough

This helps you learn faster and become independent.
