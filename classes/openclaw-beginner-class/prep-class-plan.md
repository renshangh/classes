# Prep Class Plan (45-60 min): Logistics + Readiness

## Objective
Ensure every student is technically and operationally ready before Class 1.

## Outcomes
By end of Prep Class, each student can:
- Confirm device + account access requirements
- Join required channels (live class + Telegram)
- Run one basic OpenClaw interaction successfully
- Follow support/escalation process for homework

## Agenda

### 0-10 min — Orientation
- Program overview (Prep + 3 classes)
- How learning will work (live + async support)

### 10-25 min — Access and Device Checklist
- Device/browser/network readiness
- GitHub access check
- OpenClaw access check
- Telegram group confirmation

### 25-40 min — Live Verification
- Everyone runs a minimal "hello workflow"
- Verify expected output format
- Resolve setup blockers in real time

### 40-50 min — Support Operations
- Explain stuck protocol:
  - Goal
  - Attempted steps
  - Actual output/error
  - Expected output
  - Hypothesis
- Explain help ladder (hint → scaffold → full walkthrough)

### 50-60 min — Closeout
- Publish checklist completion status
- Confirm readiness for Class 1

## Required artifacts
- onboarding-checklist.md
- stuck-protocol.md
- support-channel-rules.md
