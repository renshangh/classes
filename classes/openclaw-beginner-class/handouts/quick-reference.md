# OpenClaw Beginner Quick Reference

## Prompt Formula
Use this structure for better results:
1. **Goal:** what success looks like
2. **Context:** relevant background only
3. **Constraints:** time, format, tools, tone
4. **Output format:** exact structure wanted
5. **Verification:** how to check quality

Example:
- Goal: Create class notes for a 1-hour beginner lesson.
- Constraints: concise, practical, markdown bullets.
- Output: agenda + exercise + homework.
- Verification: include measurable outcomes.

## Reliability Checklist
- Is the task specific?
- Did you provide needed files/context?
- Are output requirements explicit?
- Is there a quality check step?
- Did you test with one real example?

## The Performance Equation
**Agentic AI performance = Model quality × Context quality × Tooling × Budget (tokens/latency)**

If quality is low, inspect in this order:
1. Context quality
2. Task structure
3. Tool reliability
4. Model selection
5. Budget/latency constraints

## Teaching Tip
Teach students to iterate quickly:
- Round 1: get a rough result
- Round 2: tighten format and constraints
- Round 3: add verification and edge cases
