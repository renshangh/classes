# OpenClaw Beginner Class (3 x 1 hour)

This is a 3-session beginner course designed for live online teaching.

## Course Goal
By the end of class 3, students can:
- Set up OpenClaw basics
- Run practical assistant workflows
- Build a simple repeatable agent workflow using skills, memory, and tooling

## Class Structure
- **Class 1:** Foundations + first wins
- **Class 2:** Practical workflows + tool usage
- **Class 3:** Agentic design + capstone demo

## Recommended Delivery Rhythm (60 min)
- 0-10 min: recap + outcomes
- 10-25 min: concept teaching
- 25-45 min: live demo / guided practice
- 45-55 min: challenge exercise
- 55-60 min: Q&A + homework
