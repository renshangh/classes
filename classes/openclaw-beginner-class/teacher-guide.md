# Teacher Guide (Instructor-Only)

## Purpose
This guide is for instructors delivering the OpenClaw Beginner Program.

## Delivery model
- Prep Class (45-60 min): logistics and readiness
- Class 1-3 (60 min each): foundations, workflows, capstone

## Instructor responsibilities
- Run setup/readiness checks before Class 1
- Lead live demos and pace guided practice
- Apply hint ladder support for homework
- Collect and triage recurring student blockers

## Class operations
- Live platform: Zoom/Meet (final selection pending owner approval)
- Async communication: Telegram (required)
- Backup plan: recorded recap + Telegram Q&A if live issue occurs

## Homework support protocol
1. Student submits stuck report (goal/attempt/output/expected/hypothesis)
2. Instructor responds with Level 1 hint first
3. Escalate to scaffold then full walkthrough if needed

## Evaluation checklist
- Can student create structured prompts?
- Can student run tool-enabled workflow?
- Can student explain one failure + recovery?
- Is capstone reproducible?

## Release workflow
- All material changes via PR
- Owner review required before merge
