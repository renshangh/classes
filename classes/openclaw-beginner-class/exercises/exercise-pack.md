# Exercise Pack (3 Classes)

## Class 1 Exercise — First Useful Output (15 min)
### Task
Create a folder for a mini project and generate a one-page project brief.

### Success Criteria
- Folder created
- Brief contains: objective, audience, deliverables, timeline

### Reflection
What part required clarification in your prompt?

---

## Class 2 Exercise — Multi-step Workflow (20 min)
### Task
Run a 4-step workflow:
1. Research a topic
2. Summarize findings
3. Create checklist
4. Save final markdown file

### Success Criteria
- All 4 steps completed
- Output follows requested format
- At least one revision based on quality check

### Reflection
Where did the workflow fail first, and how did you fix it?

---

## Class 3 Exercise — Capstone Build (25 min)
### Task
Design one repeatable agent workflow for your real life/work.

### Required Sections
- Workflow name
- Inputs
- Steps
- Tools
- Expected output
- Verification checks
- Failure recovery plan

### Success Criteria
- Reproducible by another student
- Includes at least 2 verification checks
- Includes one cost/latency optimization idea

---

## Answer Guide (Instructor)
- Strong submissions are specific, testable, and reusable.
- Weak submissions are vague, tool-free, or missing validation.
- Coach toward explicit constraints and measurable outputs.
