# Student Post-Class Handout

## Purpose
Use this after each class to reinforce learning and complete homework effectively.

## After-class routine (15-20 min)
1. Review class notes
2. Re-run the in-class workflow once
3. Improve one part (prompt, structure, or verification)
4. Save your updated version

## Homework quality checklist
- [ ] Clear goal and constraints
- [ ] Expected output format defined
- [ ] At least one verification check included
- [ ] One retry/improvement documented

## Reflection prompts
- What failed first?
- What changed on retry?
- What would you automate next?

## When asking for help
Use the stuck protocol and include exact output/error text.
