# Student Pre-Class Handout

## Purpose
Use this before the program starts so you can fully participate in hands-on activities.

## Checklist (complete before Prep Class)
- [ ] Laptop/desktop ready with stable internet
- [ ] OpenClaw access confirmed
- [ ] GitHub account ready
- [ ] Telegram installed and group joined
- [ ] Camera/mic tested for live class

## Bring to Prep Class
- One real task you want AI to help with
- One example of where you got stuck using AI before

## Readiness test
Before class, run one simple interaction and confirm you receive a response.

## If you are blocked
Post using this format:
1. Goal
2. What you tried
3. What happened
4. What you expected
5. Your best guess
