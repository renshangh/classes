# Student Handbook (Handout)

Welcome to the OpenClaw Beginner Program.

## Program at a glance
- Prep Class: get your setup ready
- Class 1: first practical result
- Class 2: multi-step workflows
- Class 3: capstone workflow

## What you need
- Computer + stable internet
- OpenClaw access
- Telegram access
- GitHub account

## How to ask for help (Stuck Protocol)
When you are blocked, share:
1. Goal
2. What you tried
3. What happened (output/error)
4. What you expected
5. Your best guess

## How support works
- First response: hint
- Next: scaffold/template
- Final: full walkthrough

## Homework expectations
- Be specific in prompts
- Follow output format requested
- Include one verification check
- Note one improvement after retry

## Success goal
By the end, you should build one repeatable workflow you can use in real life/work.
