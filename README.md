# Classes Repository

This repository contains curriculum materials for AI-enabled classes, starting with the **OpenClaw Beginner Program**.

## Current Program
### OpenClaw Beginner Program (Prep + 3 Classes)
Location:
- `classes/openclaw-beginner-class/`

Program structure:
- Prep Class (logistics + readiness)
- Class 1 (foundations + first practical win)
- Class 2 (workflow + tool usage)
- Class 3 (agentic loop + capstone)

## Material Organization
Inside each program folder:
- `student/` — student-facing materials (ordered reading flow)
- `teacher/` — instructor-facing guides and project plan
- `classes/` — per-session plans (`prep-class`, `class-1`, `class-2`, `class-3`)
- `exercises/` — exercise packs
- `shared/` — shared framework notes

## Language Support
Core materials are provided in:
- English (`*.en.md`)
- Chinese (`*.zh.md`)

## Workflow
- Work is done via **feature branches + PRs**
- Owner review/approval required before final merge
- Materials are iterated continuously based on teaching feedback

## Quick Start
1. Open `classes/openclaw-beginner-class/student/1-student-handbook.en.md` (or `.zh.md`)
2. Continue with `student/2-syllabus.en.md` (or `.zh.md`)
3. Use pre/post class handouts and session folders during delivery
