# Post‑Class Reinforcement Loop Template (Epic B‑2)

## Goal
Provide a consistent structure for post‑class follow‑up that helps students consolidate learning, reflect on challenges, and plan next steps.

## Template Sections

1. **Summary of Key Takeaways**
   - Bullet list of the most important concepts covered.
   - One‑sentence “big picture” statement.

2. **Checkpoint Review**
   - List each checkpoint from the lesson with the student’s self‑assessment (✔️/⚠️) and any notes.
   - Example:
     ```markdown
     - Checkpoint 1 – Prompt Engineering: ✔️
       - Note: Needs more practice with temperature settings.
     ```

3. **Homework / Exercises**
   - Brief description of the assignment.
   - Submission deadline (ISO date).
   - Link to the GitHub folder or Google‑Drive location.

4. **Common Blockers & Resolutions**
   - Summarize any recurring issues (e.g., API key missing, environment setup).
   - Provide a short resolution or a link to a troubleshooting guide.

5. **Next Lesson Preview**
   - One paragraph preview of the upcoming lesson.
   - Suggested pre‑reading or warm‑up activity.

6. **Feedback Prompt**
   - Quick 3‑question survey (e.g., rating 1‑5, what was most useful, one improvement suggestion).
